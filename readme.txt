A WordPress plugin to use recharge feacility from Valuetopup Plugin by Prepay Nation. 
 
== Description ==
 
Start offering Mobile TopUp & Airtime Recharge Services on your website by installing this plugin in a few simple steps and become our business partner.

This plugin allows your website customers to buy Mobile TopUp or Airtime recharges across different countries around the globe for multiple Mobile Operators in just a few steps.

Register on https://valuetopup.com/signup/index to discuss & know more about the business opportunities we offer to our partners.
 
== Installation ==
 
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
 
== Frequently Asked Questions ==
 
= How do I use this plugin? =
 
* Select country and Enter mobile number
* Our system will auto detect the operator and You will also have the option to choose the correct operator.
* Enter Amount to recharge
* Make Payment using Credit & Debit Card 
* Recharge successful
 
= How to uninstall the plugin? =
 
Simply deactivate and delete the plugin.
